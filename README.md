<현재 진행상황><br>
사용자 정보 - 완료<br>
게시판 관리 - 일부 완료했으나 검색기능 및 상세정보 보기 추가할 예정<br>
실시간 채팅 - 완료<br>

<테스트 방법><br>
- 사용자 정보<br>
1. 관리자로 로그인(/admin으로 이동 됨)<br>

- 게시판 관리<br>
1. 사용자로 로그인 후 /api.auth/board 접속<br>
2. 관리자로 로그인 후 게시판 관리 클릭<br>

- 실시간 채팅<br>
1. 크롬으로 사용자 로그인 후 /api.auth/livechat 접속<br>
2. 다른 브라우저로 관리자 로그인 후 실시간 채팅에 접속<br>
3. livechat에 접속되어 있는 유저가 있다면 연결 중이 뜨고 접속 유저의 정보가 뜸<br>
4. 클릭 후 실시간 채팅<br>
